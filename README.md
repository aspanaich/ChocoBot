# ChocoBot

In Memoriam 🍫

## Installation

1. Grab Crystal from <https://crystal-lang.org/install/>
2. Clone the repo
3. Run `shards install` and `shards build`
4. Create `.env` in the project root with the content `DISCORD_TOKEN = your-token-here`
5. Run `bin/ChocoBot`
